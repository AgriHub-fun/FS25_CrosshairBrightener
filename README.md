# FS25_CrosshairBrightener
Mod: Brightening the crosshairs from 25% alpha to 100% alpha

<img width="256" height="256" alt="icon_crosshairBrightener" src="https://github.com/user-attachments/assets/187994a7-2407-468d-b25d-ed9e4845b270" />

## English

Brightening the crosshairs from 25% alpha to 100% alpha, as using FSR/DLSS causes a transparency error that makes the crosshairs almost impossible to see.

## Deutsch

Aufhellen des Fadenkreuz von 25 % Alpha auf 100 % Alpha, da bei Verwendung von FSR/DLSS ein Transparenzfehler auftritt wodurch man das Fadenkreuz kaum noch sehen kann.
